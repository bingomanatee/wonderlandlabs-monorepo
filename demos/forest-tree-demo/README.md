A visual demonstration of the Forestry4 state management library using a literal forest metaphor. This demo shows an interactive tree with branches and leaves that respond to wind forces, all managed by Forestry4's Forest and Branch architecture.

## Features

- **Interactive Tree Visualization**: SVG-rendered tree with curved branches and animated leaves
- **Wind Simulation**: Mouse movement creates wind forces that affect the tree
- **Forestry4 Integration**: Uses Forest as root state and branches for wind force management
- **Real-time Animation**: Smooth 20 FPS animation loop with wind effects
- **Season Controls**: Switch between seasons (future feature for leaf color changes)
- **Force-Directed PhysicsMgr**: Branches curve upward naturally and respond to wind
